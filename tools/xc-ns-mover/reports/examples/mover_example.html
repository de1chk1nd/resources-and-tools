⚠ Important — CNAME & ACME / Let's Encrypt Warning

When a load balancer is moved to a new namespace (or reverted back), F5 XC assigns a new CNAME (host_name). If you are using Let's Encrypt (auto-cert), the ACME challenge domain will also change.

  • Update your DNS records to point to the new CNAME — see the DNS Changes section below for a full list.
  • If the LB was reverted after a failed move, the CNAME in the source namespace may differ from the original — check the value carefully.
  • Let's Encrypt certificates will be re-issued automatically once DNS propagates, but there may be a brief period without a valid certificate.
